# Description
A movie sites which display a list of updated movie that exist within tmdb API data

# Features
  1. Register and Login
  2. Movie List
  3. Detail Movie
  4. Watch Later
  5. Watch Trailer
  
# Technologies 
  1. React JS
  2. Tailwind CSS
  3. firebase-auth
  4. redux-toolkit
